Steps for building a new bootstrap page

  1. Download bootstrap
  2. capture a "skeleton" HTML page
  3. create a custom.css file and link to it
  4. envelope contents of the body in a "container" div
  5. remember to use Validation tools (posted to the resources page on the course site) Firefox/Firebug or other Developer tools (e.g. in Chrome) to assist you in debugging
  6. read through bootstrap documentation to get ideas and ENJOY
