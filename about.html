BootStrap Project

This is a course project of Web Development and Programming.
xHTML, CSS and Frameworks: Twitter Bootstrap

Course site

Project Requirements

  1. The pages should have at least three layout items (e.g. header, menu bar or "navbar", main content, etc) which do NOT use absolute positioning.
  2. Use a custom CSS file (typically called "custom.css"). In other words, if you would like to use a Twitter Bootstrap theme, that is fine; but please do customize it.
  3. The content pages must include at least one of each of the following. You may use Bootstrap styles or original styles or modify the Bootstrap styles:
    • Include at least one styled list with original bullets.
    • Include at least one styled table.
    • Include at least one image on every content page. The image should float in a similar place on each page.
  4. Define CSS rules in the "custom.css" file as follows:
    • at least three original styles with HTML selectors
    • at least three original styles with class selectors
    • at least three additional styles using "pseudo-classes" (e.g. styles for a specific ID selector)
  5. Use at least one example of a JavaScript plugin such as a modal,carousel or other "special effect" that uses Bootstrap's JavaScript. We will look at several examples in class. You do not need to modify the JavaScript.
  6. Use consistent navigation so that the users can get to all of the pages from every page. (Please note that if you prefer to use Bootstrap's tab-content layout, that each "tab" would "count" as a page.)
  7. Use a consistent heading (either as text or a graphic) with brief information about the pages (e.g. "Poems by William Carlos Williams").
  8. Use a background color or background image on every page. (They don't have to be the same for every page.)

Build a unified web site consisting of one opening page and at least three pages on a unified topic.

 

Work with Twitter Bootstrap and to exploit CSS as fully as possible to accomplish the following goals so that all three pages look alike outside of the specific content.

 

The goal is to do this with Bootstrap and CSS.
